WordPress 常用插件

2014年 4月28日

我的主博客 使用的一些 WordPress 插件，留此存档。

  • Askimet: 消除垃圾评论，系统自带。
  • Archive: 早年写的一些低质量文章，自己不想删，那么就选择归档吧。
  • Broken Link Checker: 写了八年博客了，有些链接已经失效，这次改版用这个插件顺手修复了。
  • Easy FancyBox: 有需要放大显示的图片，用这个比较省事。
  • Google Analytics for WordPress: Google 统计。
  • Google XML Sitemaps: 网站地图生成器。
  • Internal Link Building: 内部链接生成。
  • Jetpack by WordPress.com: 这个 WordPress 自家出的插件包里有很多有用插件，比如 Markdown.
  • Last Updated Posts Widget: 显示最近修改文章的插件。
  • M7 Go Top: 返回顶部。
  • Multicolumn Category Widget: 多栏显示分类目录。
  • RunCode: Run html code in a textarea.
  • WP-Bigfoot: Instapaper 风格的注释，我见过的最佳注释呈现方式。
  • WP-UTF8-Excerpt: 在首页显示摘要时，保留文字格式，同时保证 RSS 全文输出，作者貌似是网上小有名气的小骆驼商队。
  • WP Super Cache: 静态页面生成，加快网站访问速度。
  • 多说: 国内 Disqus 又水土不服了，只好用多说，好在多说做得还不错。
